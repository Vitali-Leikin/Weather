

# Weather
### Traditional Weather app.

* **Features**:
  * Display the weather for the current location using CoreLocation
  * Search by city name
  * Saving current the weather
  * Saving search results

___

* **Knowledges**:
  * Rest API
  * Bindable values 
  * UIKIT
  * Codable
  * CoreLocation
  * UserDefaults
  * MVC
